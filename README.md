# 🐱 Hidden Friends — 3D Adventure Game

A cozy 3D hide-and-seek adventure built in Unity with C#.
Play as a little cat searching for 10 hidden friends across a lively scene.

🎮 Play it now: https://sazaki-0.itch.io/hidden-friends

---

## 🎮 About the Game

Step into the paws of a small curious cat playing hide-and-seek with friends!
Explore a calm, charming environment in first-person view, find your hidden friends one by one, and watch them magically teleport back to the gathering table.
Watch out for obstacles along the way — they can take away your lives!

When all 10 friends are collected, the game celebrates with fireworks, victory sound effects, and a win screen before returning to the main menu.

---

## ✨ Features

- 🐾 First-person 3D movement — walk, run, jump, and interact
- 👥 Friend collection system — find all 10 hidden friends to win
- ❤️ Health & lives system — avoid obstacles or lose a life
- 🔁 Respawn system — return to the spawn point when defeated
- 🎵 Audio design — jump SFX, collect SFX, damage SFX, background music
- 🎆 Victory VFX — fireworks effect when all friends are found
- 🪑 Gathering table — friends teleport here when collected
- 📊 HUD UI — live counter for friends collected & lives remaining
- 🎬 Scene transitions — Main Menu → Gameplay → Win Screen → Main Menu

---

## 🛠️ Built With

- Unity (3D Engine)
- C# (Game logic & scripting)
- ShaderLab + HLSL (Custom shaders)
